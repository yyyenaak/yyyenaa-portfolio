    Farmy의 메인 인덱스는 스마트팜의 핵심 환경 데이터를 직관적으로 시각화하고 간편하게 제어할 수 있도록 설계되었습니다.

      상단: 온도, 습도, 조도, 토양 수분 등 주요 환경 데이터를 실시간으로 제공하여, 작물의 상태를 한눈에 확인하고 신속하게 관리할 수 있습니다.
      중앙: 네 가지 기능 버튼이 직관적인 아이콘과 함께 배치되어, 각 제어 시스템으로 쉽게 이동할 수 있습니다.
    • 온도(온도계 아이콘): 최적의 온도를 설정 및 조절
    • 습도(팬 아이콘): 적정 습도 유지 및 모니터링
    • 조도(전구 아이콘): 조명 시스템 제어로 광합성 지원
    • 토양 수분(식물+물방울 아이콘): 토양 수분 상태 확인 및 자동 공급
      하단: Chat me! (ChatGPT 연동) 기능을 통해 실시간 데이터 분석 및 최적의 환경 설정을 제안하여, 스마트한 작물 관리를 지원합니다.

    메인 화면은 녹색과 파란색 그라데이션을 활용해 자연 친화적인 감성을 강조하며, 심플하고 직관적인 레이아웃으로 누구나 쉽게 사용할 수 있도록 설계되었습니다.

    Controll Pop-up

    Farmy의 팝업은 사용자가 작물 생장에 중요한 온도, 습도, 조도, 토양 수분을 세밀하게 조절할 수 있도록 설계되었습니다. 메인 화면에서 해당 항목을 클릭하면 팝업이 중앙에 표시되며, 배경을 흐리게 처리하여 사용자가 선택한 센서의 데이터 조절에 집중할 수 있도록 최적화되었습니다.

      메인: 온도계 아이콘과 함께 "Temperature" 제목이 표시되며, 현재 설정된 온도가 중앙에 나타납니다. 사용자는 온도 슬라이더를 통해 직관적으로 원하는 온도를 설정할 수 있습니다. 또한, 선택한 항목에 따라 "습도", "조도", "토양 수분"으로 변경되어 해당 센서를 조절할 수 있습니다
    • 온도 범위: 0°C부터 30°C까지 제공되며, 다양한 작물의 생육 조건에 맞춰 조절할 수 있습니다.
    • 슬라이더 디자인: 간결하고 직관적인 인터페이스로 설계되어, 사용자가 불편함 없이 쉽게 조작할 수 있습니다.
      우측 상단: 닫기(X) 버튼을 배치하여 팝업을 빠르게 종료하고, 메인 화면으로 원활하게 돌아갈 수 있도록 설계되었습니다.

    이는 불필요한 요소를 최소화하고 핵심 기능에 집중하여, 간편하면서도 정밀한 환경 제어 기능을 제공합니다.

    Farmy GPTs

    Farmy GPTs는 스마트팜과 식물 관련 정보를 쉽고 빠르게 탐색하고 학습할 수 있도록 설계된 AI 도우미입니다. 메인 화면에서 식물 성장과 환경 조절에 대한 질문을 선택하면 AI가 해당 정보를 제공하며, 사용자는 실시간으로 스마트팜 관리에 필요한 데이터를 얻을 수 있습니다.

      설계: 상단에는 식물 아이콘과 함께 "Farmy"라는 제목이 표시되어 브랜드 정체성을 강조하고, 직관적인 UI를 통해 사용자가 손쉽게 정보를 탐색할 수 있도록 설계되었습니다.
    • 질문 선택 방식: "토마토 성장의 적정 온도는?", "강낭콩 성장의 적정 습도는?" 등과 같은 버튼형 질문을 제공하며, 클릭하면 AI가 적절한 답변을 제공합니다.
    • 메시지 입력창: 사용자가 자유롭게 질문을 입력할 수 있도록 하단에 배치되었으며, 첨부 파일 업로드 및 음성 입력 기능도 지원됩니다.
    Farmy GPTs
      학습 방법: 스마트팜 운영에 필요한 온도, 습도, 토양 정보를 AI를 통해 학습하며, 다양한 식물물 데이터와 논문 PDF를 분석하여 사용자가 식물 성장에 대한 이해도를 높일 수 있도록 구성되었습니다. 프롬프트 엔지니어링 기법을 활용해 심화 학습을 진행하였습니다.
    • Zero-Shot Prompting: 사전 학습 없이도 즉각적인 답변을 생성하여 스마트팜과 식물 관련 정보를 제공합니다.
    • Chain of Thought: 논리적 사고 과정을 적용해 단계별 추론을 수행하며, 보다 정확하고 신뢰도 높은 답변을 제공합니다.
    • Automatic Prompt Engineering: 사용자의 질문을 분석하고 최적화하여, 자연스럽고 정확한 답변을 생성합니다.

    불필요한 정보는 최소화하고 핵심 내용과 파생 정보에 집중하여, 간편하면서도 효과적인 정보 탐색이 가능하도록 최적화되었습니다. 또한, 제약조건 설정을 통해 식물과 스마트팜 관련 질문 외에는 입력할 수 없도록 설정하였습니다.

    앱 디자인 후 WebView 방식으로 개발을 계획했으나, 실시간 데이터 제공을 위해 1차로 HTML, CSS, JS 기반으로 개발하였습니다. 이후, 안드로이드 앱 개발을 위해 XML과 Java를 활용한 2차 개발을 진행하였으며, 실시간 데이터 UI 및 제어 토글 화면을 수정하여 최종 디자인과 개발을 완료하였습니다.

    • 스마트팜 개발을 위해 관련 모듈을 조립하고, 자동/수동 모드 전환이 가능한 아두이노 코드를 작성하였습니다. 아두이노와 연동하여 센서 데이터를 실시간으로 송수신하고, 코드 최적화를 통해 안정적인 데이터 처리가 가능하도록 구현하였습니다.

    Farmy GPTs는 스마트팜과 식물 관련 정보를 쉽고 빠르게 탐색하고 학습할 수 있도록 설계된 AI 도우미입니다. 메인 화면에서 식물 성장과 환경 조절에 대한 질문을 선택하면 AI가 해당 정보를 제공하며, 사용자는 실시간으로 스마트팜 관리에 필요한 데이터를 얻을 수 있습니다.

      설계: 상단에는 식물 아이콘과 함께 "Farmy"라는 제목이 표시되어 브랜드 정체성을 강조하고, 직관적인 UI를 통해 사용자가 손쉽게 정보를 탐색할 수 있도록 설계되었습니다.
    • 질문 선택 방식: "토마토 성장의 적정 온도는?", "강낭콩 성장의 적정 습도는?" 등과 같은 버튼형 질문을 제공하며, 클릭하면 AI가 적절한 답변을 제공합니다.
    • 메시지 입력창: 사용자가 자유롭게 질문을 입력할 수 있도록 하단에 배치되었으며, 첨부 파일 업로드 및 음성 입력 기능도 지원됩니다.
    Farmy GPTs
      학습 방법: 스마트팜 운영에 필요한 온도, 습도, 토양 정보를 AI를 통해 학습하며, 다양한 식물물 데이터와 논문 PDF를 분석하여 사용자가 식물 성장에 대한 이해도를 높일 수 있도록 구성되었습니다. 프롬프트 엔지니어링 기법을 활용해 심화 학습을 진행하였습니다.
    • Zero-Shot Prompting: 사전 학습 없이도 즉각적인 답변을 생성하여 스마트팜과 식물 관련 정보를 제공합니다.
    • Chain of Thought: 논리적 사고 과정을 적용해 단계별 추론을 수행하며, 보다 정확하고 신뢰도 높은 답변을 제공합니다.
    • Automatic Prompt Engineering: 사용자의 질문을 분석하고 최적화하여, 자연스럽고 정확한 답변을 생성합니다.

    불필요한 정보는 최소화하고 핵심 내용과 파생 정보에 집중하여, 간편하면서도 효과적인 정보 탐색이 가능하도록 최적화되었습니다. 또한, 제약조건 설정을 통해 식물과 스마트팜 관련 질문 외에는 입력할 수 없도록 설정하였습니다.
